# Politik

## Installation

### Frontend

1. Move to `frontend` directory:
   `cd frontend`
2. Add your environmental variables in `.env` file as shown in `.env.example` file.
3. Install dependencies:
   `npm install`
   or
   `yarn install`
4. Run development server:
   `npm run dev`
   or
   `yarn dev`

**Additional scripts:**
Build application:
`npm run build`
or
`yarn build`

Start built application:
`npm run start`
or
`yarn start`
